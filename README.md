# Sunny Side — Static Restaurant Website

This repository contains a small static restaurant website built with plain HTML, CSS and JavaScript and prepared for deployment to GitHub Pages.

## What’s included

- `index.html`: main site markup
- `css/styles.css`: site styles and responsive layout
- `js/script.js`: small scripts (mobile nav, contact form, image fallback loader)
- `images/`: dish photos and placeholders (used by the menu cards)
- `.github/workflows/gh-pages.yml`: GitHub Action to publish the site to the `gh-pages` branch

## Features

- Responsive design for mobile and desktop
- Menu with dish images and fallback filename logic
- Simple contact form validation (client-side)

## Images and menu items

- Place images in the `images/` folder. Example filenames referenced by the site:
  - `breakfast_platter.png`
  - `pancakes.png`
  - `avocado toast.png` (contains a space in the filename)
  - `burger.jpg`
  - `pasta.png`
  - `garden_salad.jpg`
